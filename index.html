ImageBind

This is an interactive demo for the work ImageBind. Imagebind binds together six modalities: images, video, audio, text, depth, thermal, and IMU measurements. It binds these different modalities through the inherent alignments between each of them with images, achieving an aligned embedding space between all of them. Please refer to the authors' paper for more details!

Below is a cross-modal retrieval system built using the ImageBind model. Get started by entering queries, or use the navigation bar to see examples and the results of our experiments. The code for the retrieval system and experiements can be found here: https://github.com/arul-aravinthan/ImageBind-Demo

Examples

Here we are retrieving cat images, with audio of a cat meowing. However, the cats are shown in a variety of scenarios. If we want to get pictures of sleeping cats specifically...

We can add an image of a bed to the query. Now the results show cats sleeping in beds!

Here we are retrieving images of umbrellas in the rain. The umbrellas are present, but the rain is not very clear. In order to show both the rain and the umbrellas in the results...

We can add a rain image to the query. Now the results show umbrellas and more visble rain!
